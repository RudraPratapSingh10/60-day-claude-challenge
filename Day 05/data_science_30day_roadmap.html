30-day data science roadmap

Milestone — Python & data wrangling mastery
Daily tasks
  • Day 1 Review NumPy arrays, indexing & broadcasting
  • Day 2 Pandas: DataFrames, read_csv, head/info/describe
  • Day 3 Data cleaning — nulls, duplicates, dtypes
  • Day 4 GroupBy, merge, pivot tables
  • Day 5 Matplotlib basics — line, bar, scatter
  • Day 6 Seaborn — heatmaps, pairplots, distributions
  • Day 7 Mini project + review weak areas
Resources
  • Kaggle — Pandas micro-course (free)
  • NumPy docs quickstart tutorial
  • "Python for Data Analysis" — Wes McKinney (ch. 1–5)
  • Dataset: Titanic (Kaggle)
  • Corey Schafer — Pandas playlist
P1
Titanic data exploration report
Load the Titanic dataset, clean it, explore survival rates by gender/class/age, and produce 5 meaningful visualizations with written insights. Submit as a Jupyter notebook.
Milestone — Build & evaluate your first ML models
Daily tasks
  • Day 8 ML fundamentals — supervised vs unsupervised
  • Day 9 Linear regression from scratch + scikit-learn
  • Day 10 Logistic regression & classification metrics
  • Day 11 Decision trees & Random Forest
  • Day 12 Train/test split, cross-validation, overfitting
  • Day 13 Feature engineering & encoding categoricals
  • Day 14 Model comparison + project day
Resources
  • Kaggle — Intro to ML course (free)
  • StatQuest — ML fundamentals (YouTube)
  • Scikit-learn user guide (online docs)
  • Dataset: House Prices (Kaggle)
  • Sentdex — scikit-learn series
P2
House price prediction model
Use the Ames Housing dataset to predict sale prices. Engineer features, train linear regression and random forest models, compare RMSE scores, and write a 1-page summary of findings.
Milestone — EDA + advanced ML + real competition
Daily tasks
  • Day 15 EDA deep-dive: outlier detection, correlations
  • Day 16 Hyperparameter tuning — GridSearchCV
  • Day 17 XGBoost / Gradient Boosting intro
  • Day 18 Pipelines in scikit-learn (end-to-end)
  • Day 19 Natural Language Processing basics — NLTK
  • Day 20 SQL for data science — joins, subqueries
  • Day 21 Kaggle competition submission day
Resources
  • Kaggle — Intermediate ML course
  • XGBoost official docs & examples
  • Mode SQL Tutorial (free online)
  • Dataset: any active Kaggle competition
  • Krish Naik — end-to-end ML projects
P3
Kaggle competition entry
Enter a beginner-friendly Kaggle competition (e.g. Spaceship Titanic). Build a full scikit-learn pipeline, tune with GridSearchCV, submit predictions, and document your approach in a public notebook.
Milestone — Portfolio-ready capstone project
Daily tasks
  • Day 22 Choose capstone topic & gather dataset
  • Day 23 Full EDA with business question framing
  • Day 24 Feature engineering + model training
  • Day 25 Model evaluation & error analysis
  • Day 26 Build a simple Streamlit dashboard
  • Day 27 Write project README + clean notebook
  • Day 28–30 Push to GitHub, share on LinkedIn, reflect
Resources
  • Streamlit docs — Get started guide
  • "How to write a great data science README"
  • Towards Data Science (Medium publication)
  • UCI ML Repository or Kaggle datasets
  • GitHub + nbviewer for sharing notebooks
P4
Capstone — end-to-end data science project
Pick a real-world dataset you care about (sports, health, finance, etc.), answer a meaningful question with data, build a polished ML model, and deploy a Streamlit app. This is your portfolio centrepiece.

Final outcome — what you'll have after 30 days

4 GitHub projects Kaggle profile Live Streamlit app pandas & numpy fluency scikit-learn pipelines XGBoost experience SQL basics Portfolio-ready notebook
