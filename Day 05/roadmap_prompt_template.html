Prompt A vs Prompt B — what changed?
Prompt A — generic
Create a 30-day learning roadmap. Include weekly milestones, daily tasks, resources, projects, and final outcome.
No context = generic output. Could be for anyone, so it's useful to no one in particular.
Prompt B — personalized
Create a 30-day roadmap. Context: - Role: Student - Skills: Basic C++, Python - Goal: Job-ready in Data Analytics - Time: 5 hrs/day - Level: Beginner - Style: Videos + Projects
6 context variables = output tailored to your exact situation, skills, and career goal.

Full prompt preview — what to copy into Claude
Create a 30-day learning roadmap for [YOUR TOPIC]. Context: - Current situation: [Student / Professional / Career switcher] - Current skills: [What you already know] - Goal: [What job-ready outcome you want] - Available time: [X hours per day] - Experience level: [Beginner / Intermediate] - Learning style: [Videos / Projects / Reading] - Location: [Your country or job market] ← optional but powerful Include: - Weekly milestones - Daily tasks with time blocks - Free resources (videos + tools) - Hands-on projects for each week - Final outcome and portfolio plan Make it practical and beginner-friendly.
Why each context variable matters
Your roleSets pace, budget & goal type
Current skillsSkips basics you already know
Specific goalShapes the final week's content
Daily timeCreates realistic daily blocks
Experience levelAdjusts depth and speed
Learning stylePicks right resource types
LocationUses locally relevant datasets & tools
